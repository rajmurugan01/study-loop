Artifact 6: Google Calendar setup guide
How to structure {{STUDENT_NAME}}'s exam calendar so the routine reads it cleanly.
Calendar
Create a new calendar (not an event in your existing calendar) named:
"{{CALENDAR_NAME}}"
Exact name matters — the routine prompt references it by name. If you choose a different name, update the prompt accordingly.
Event format
Event title
Format: [Subject] Exam: [Topic name]
Examples:
Maths Exam: Term 1 — Products, Factors and Surds
English Exam: Romeo and Juliet essay
Science Exam: Biology — Cells and Heredity
Maths Exam: Half-Yearly
Event date and time
Use the actual exam date. Set as all-day event unless you know the exact time — exact time helps the routine flag "morning of exam" vs "after school" timing if relevant for your prep planning.
Event description (this is the important field)
Include in the description, on separate lines:
Scope: chapters or topics (e.g. "Chapters 4.03 to 4.16" or "R&J Acts 1-3, themes of fate and conflict")
Weighting: percentage of overall subject mark (e.g. "30% of Term 1 mark")
Duration: how long the exam is (e.g. "45 minutes" or "1 hour 30 minutes")
Format: paper, practical, oral, etc. (e.g. "Written paper" or "Practical + written")
Anything else relevant: calculator allowed? Open book? Special notes from teacher?
Sample event
Title: Maths Exam: Term 1 — Products, Factors and Surds

Date: Sat 24 May 2026, all day

Description:
Scope: Chapters 4.03–4.16 (Algebraic techniques and Surds)
Weighting: 30% of Term 1 mark
Duration: 45 minutes
Format: Written paper
Notes: Calculator allowed. No formula sheet.
For exam dates you don't have yet
Two approaches:
Approach A: estimated placeholders
Create the event with:
Title: "[Subject] Exam: TBC"
Date: best guess based on the school's term calendar (typically last 2 weeks of term)
Description: include line "Date is estimated — confirm with school"
The routine will read these and use them, flagging in the Dad email that the date is estimated.
Approach B: leave them out
If you have no idea when an exam will be, just don't create the event. The routine will note that the subject has no upcoming exam in the next 90 days, and recommend maintenance-level revision until you add a date.
When to update
School publishes exam timetable: update placeholder events with real dates and any scope changes
Mid-term tests, quizzes, surprise assessments: add as you find out
Teacher changes scope: edit the event description, save
The routine reads the calendar fresh each Sunday — no need to do anything else when you change something.
